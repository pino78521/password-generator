# Password Generator (GUI Version)

## Description
This is a graphical version of the password generator built using Python's `tkinter` library.
It generates passwords with:
- 6 letters (random upper/lower case)
- 2 numbers
- 1 special character

## How to Install and Run

### Step 1: Make sure Python is installed
Download and install Python from:
https://www.python.org/downloads/

Make sure you check the option to "Add Python to PATH" during installation.

### Step 2: Unzip the Project
Extract the `password_generator_gui.zip` file to any location on your computer.

### Step 3: Run the Program
Open a terminal (Command Prompt or PowerShell on Windows, or Terminal on macOS/Linux), and navigate to the folder:
```
cd path/to/password_generator_gui
```

Then run:
```
python main.py
```

- Click a button to generate a random password
